Welcome to the Keycloak Push MFA Extension Simulator. This tool mocks the push-based multi-factor authentication process for development and testing purposes.

Enrollment

This part simulates the process of registering a device for push-based multi-factor authentication. Users can enroll their devices to receive push notifications for MFA challenges.

Go to Enrollment
Complete Login

This simulates the completion of the login process after receiving a push notification. It handles the verification and finalization of the MFA step. When a valid request is made to /completeLogin, it triggers a push notification from Keycloak to the enrolled device.

Go to Complete Login
